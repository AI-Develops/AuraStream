# AuraStream

AuraStream is a minimalist ambient sound mixer built for focus and relaxation.
It allows users to layer different environmental sounds to create a custom
auditory backdrop directly in the browser.

## Features

- **Multi-Track Mixing:** Toggle and adjust volumes for multiple sound loops
  simultaneously.
- **Built-in Timer:** Set a Pomodoro or countdown timer to manage focus
  sessions.
- **Local Storage:** Remembers your favorite volume levels and active tracks.
- **Responsive Design:** Works seamlessly on mobile and desktop.

## Tech Stack

- HTML5 (Structure & Audio API)
- CSS3 (Custom Properties & Flexbox/Grid)
- Vanilla JavaScript (State Management & DOM Manipulation)

## Usage

Simply open `index.html` in any modern web browser. No installation or
dependencies required.
